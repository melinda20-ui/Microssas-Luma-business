# ✍️ Sualuma AI Blog

Sistema de gerenciamento de conteúdo automatizado.

## ⚙️ Configuração
1. Instale as dependências: `npm install`.
2. Configure o arquivo `.env`:
   - `NEXT_PUBLIC_API_URL`: `http://localhost:3001/api`.
   - `NEXT_API_SECRET`: Sua chave mestre.

## 🚀 Funcionalidades
- **Admin `/admin`**: Gere posts automaticamente pedindo à IA.
- **SEO Ready**: Configurado para Sitemap e Robots.txt.
- **Next.js 15**: Performance ultra-rápida.
