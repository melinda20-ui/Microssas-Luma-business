# ⚡ Sualuma MicroSaaS Core

A interface premium e futurista para atendimento e automação de negócios.

## ⚙️ Configuração
1. Instale as dependências: `npm install`.
2. Configure o arquivo `.env`:
   - `NEXT_PUBLIC_API_URL`: O link da sua `agents-api` (ex: `chat.sualuma.online/api`).

## 🎨 Design System
- **Tema**: Dark Futurista (Glassmorphism).
- **Componentes**: UI baseada em TailwindCSS v4.
- **Responsividade**: Mobile-first e Desktop-ready.

## 🚀 Funcionalidades
- **Landing Page**: Showcase dos agentes.
- **Chat Interativo**: Conexão direta com o backend de IA.
- **Dashboard**: Estatísticas e resumo de atividades.
